# 大禹流水线发布

我们认为日常和预发环境是测试环境，测试环境可直接运行流水线。

而生产环境是正式环境，正式环境的发布会直接影响用户的使用，所以发布正式环境需要严格的限制。发布正式环境需要先创建变更单。
